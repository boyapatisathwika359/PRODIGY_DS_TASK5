# 🚦 US Traffic Accident Analysis

## 📌 Project Overview

This project analyzes the **US Traffic Accidents** dataset to identify patterns related to **road conditions, weather conditions, accident severity, time of day, and accident hotspots**. The analysis was performed using Python for data cleaning, exploratory data analysis (EDA), and visualization to uncover meaningful insights that can help improve road safety and traffic management.

---

## 🎯 Objective

The primary objectives of this project are to:

- Analyze traffic accident data to identify accident patterns.
- Examine the impact of weather conditions on accidents.
- Analyze accident frequency based on the time of day.
- Identify states with the highest accident occurrences.
- Visualize accident hotspots using geographic coordinates.
- Explore relationships among different accident-related variables.

---

## 📂 Dataset

- **Dataset:** US Traffic Accidents Dataset
- **Source:** Kaggle
- **Link:** https://www.kaggle.com/datasets/sobhanmoosavi/us-accidents

## 🔍 Key Insights

- Peak traffic hours recorded the highest number of accidents.
- Severity Level 2 accidents were the most common.
- Certain states experienced significantly more accidents than others.
- Weather conditions such as Fair, Rain, Cloudy, and Fog contributed to accident occurrences.
- Urban regions showed a higher concentration of accidents.
- Reduced visibility was associated with increased accident frequency.
- Daytime accidents were more frequent than nighttime accidents.
- Geographic hotspot analysis highlighted densely populated regions with higher accident density.

---

## ✅ Conclusion

The analysis successfully identified important traffic accident patterns based on weather conditions, accident severity, geographic location, and time of day. The visualizations provide meaningful insights into accident hotspots and contributing factors. These findings can support transportation authorities, policymakers, and road safety organizations in making data-driven decisions to improve traffic safety and infrastructure planning.
